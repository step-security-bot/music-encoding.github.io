A la hora de hacer un tutorial por tu cuenta, es posible que te hayas preguntado cómo Verovio es capaz de renderizar la MEI del usuario, a pesar de que éste no haya introducido un archivo MEI completo. Esto se consigue con la propiedad "xmlFile" del paso del tutorial introducido en la última página. Esta propiedad hace referencia a un archivo MEI completo, que proporciona el contexto que necesita ir "alrededor" de la entrada del usuario para una correcta renderización. Veamos un ejemplo sencillo: 

<mei xmlns="http://www.music-encoding.org/ns/mei">
    <meiHead>
        <fileDesc>
            <titleStmt>
                <title/>
            </titleStmt>
            <pubStmt/>
        </fileDesc>
    </meiHead>
    <music>
        <body>
            <mdiv>
                <score>
                    <scoreDef>
                        <staffGrp>
                            <staffDef clef.shape="G" clef.line="2" n="1" lines="5"/>
                        </staffGrp>
                    </scoreDef>
                    <section>
                        <measure>
                            <staff n="1">
                                <layer>
                                    
                                    <note pname="c" oct="4" dur="4"/>
                                    
                                </layer>
                            </staff>
                        </measure>
                    </section>
                </score>
            </mdiv>
        </body>
    </music>
</mei>

Se trata de un archivo MEI normal, pero contiene dos instrucciones de procesamiento importantes (marcadas en rojo). El contenido de este archivo se lee en la memoria. Siempre que la entrada del usuario en el editor sea un XML bien formado, se inserta entre las dos instrucciones de procesamiento, y el archivo completo (ahora incluyendo la entrada del usuario) se valida y (si procede) se envía a Verovio para su renderización.

Lo que se almacena entre las dos instrucciones de procesamiento pretende servir como solución de muestra. Esa funcionalidad aún no está implementada, pero se añadirá pronto. Ya es posible descargar el archivo completo (incluido el contexto), para que los usuarios puedan utilizarlo como referencia futura o como modelo para su propio trabajo.
